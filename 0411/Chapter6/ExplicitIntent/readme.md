# 📱 두 개의 액티비티로 구성된 앱 예제

이 앱은 **명시적 인텐트(Explicit Intent)** 를 활용하여 두 개의 액티비티 간 전환을 구현합니다.

---

## 🎬 앱 구성

### ▶ MainActivity (첫 번째 액티비티)
- 버튼을 누르면 `SecondActivity`로 이동
- 명시적 인텐트를 통해 화면 전환 수행

#### 주요 기능:
- 화면에 버튼 UI 배치
