# 📱 생애주기(Lifecycle) 출력 애플리케이션

안드로이드 액티비티의 생애주기 메서드들을 재정의하고, 각 메서드가 호출될 때마다 로그를 통해 확인할 수 있는 간단한 애플리케이션입니다.

---

## 🔁 주요 기능

- `onCreate()`, `onStart()`, `onResume()`, `onPause()`, `onStop()`, `onDestroy()` 등의 메서드 재정의
- 각 메서드가 호출될 때 **Logcat**에 메시지 출력
- 생애주기 흐름을 눈으로 확인 가능
