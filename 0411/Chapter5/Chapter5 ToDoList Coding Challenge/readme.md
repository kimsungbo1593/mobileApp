# 할 일 목록(To-Do List) 앱

사용자가 해야 할 일을 손쉽게 기록하고 관리할 수 있는 간단한 할 일 목록 애플리케이션입니다.

## 📝 주요 기능

- 텍스트 필드에 할 일을 입력
- "추가" 버튼 클릭 시 할 일이 목록에 추가됨
- 추가된 할 일은 체크박스와 함께 표시됨

## 💡 사용 흐름

1. 사용자가 텍스트 필드에 할 일을 입력한다.
2. "추가" 버튼을 누르면 할 일이 목록에 추가된다.
3. 추가된 항목은 체크박스 형태로 보여지고, 완료 시 체크할 수 있다.
