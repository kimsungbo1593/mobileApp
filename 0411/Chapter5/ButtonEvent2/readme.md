# 🎨 버튼 클릭 시 배경색 변경 앱

이 앱은 사용자 인터랙션을 통해 **이미지 배경색을 변경**할 수 있는 기능을 제공합니다. 각 버튼을 클릭하면 이미지가 위치한 레이아웃의 배경색이 변경됩니다.

---

## 📱 앱 구성

### ✅ 주요 UI 요소

- **이미지**: 사람 일러스트가 표시된 이미지
- **버튼 4개**:
  - `Color 1`
  - `Color 2`
  - `Color 3`
  - `Color 4`
