# 안드로이드 이미지 버튼 조작 앱 만들기

각 버튼을 눌러 **이미지를 회전**시키거나, **투명도를 변경**하고,  
`ScaleType`을 변경하여 **이미지의 크기나 비율을 조정**하는 앱을 만들어보자.

## 주요 기능
- 버튼 클릭 시 **이미지 회전**
- 버튼 클릭 시 **이미지 투명도 변경**
- `ScaleType` 변경을 통한 **이미지 크기 및 비율 조정**
