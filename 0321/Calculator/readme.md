# 📱 간단한 계산기 애플리케이션 만들기

이번 실습에서는 간단한 계산기 화면을 가지는 안드로이드 애플리케이션을 작성해 봅니다.  
사용자가 두 개의 숫자를 입력하고, **더하기(+) 버튼을 눌러 결과를 확인**할 수 있습니다.

---

## 📌 기능 설명
- 두 개의 숫자를 입력할 수 있는 `EditText` 필드 제공
- `Button`을 눌러 덧셈 수행
- 결과를 `TextView`에 출력
