# 📋 컨텍스트 메뉴로 텍스트뷰 배경색 변경 앱

텍스트 뷰를 길게 눌렀을 때 나타나는 **컨텍스트 메뉴**를 통해 배경색을 바꾸는 앱입니다. 사용자는 `Red`, `Green`, `Blue` 중 하나를 선택하여 텍스트 뷰의 배경색을 변경할 수 있습니다.

---

## 🧩 기능 설명

| 항목         | 설명                                              |
|--------------|---------------------------------------------------|
| 컨텍스트 메뉴 | 텍스트 뷰를 길게 클릭하면 메뉴가 팝업됨            |
| 배경색 변경   | 선택된 메뉴 항목에 따라 텍스트 뷰의 배경색이 변경됨 |
